# 上手教程

这套技能的用法很简单：把输入直接交给它即可。

## 支持的输入

- 视频链接
- 音频链接
- 本地视频文件路径
- 本地音频文件路径
- 本地文件夹路径
- 播放列表
- 批量 URL 列表

如果是本地文件夹，脚本会自动递归查找可处理的媒体文件。

如果配置了下面三个环境变量，会自动启用 LLM 后处理：

- `LLM_BASE_URL`
- `LLM_API_KEY`
- `LLM_MODEL`

## 最常见的用法

### 1. 单个视频

输入：

```text
帮我把这个视频整理成知识笔记：https://www.youtube.com/watch?v=abc123
```

结果：

- `summary.md`
- `notes.md`
- `chapters.md`
- `knowledge.json`

适合直接放进知识库。

### 2. 本地文件

输入：

```text
整理这个本地文件：C:\Users\Administrator\Downloads\demo.mp4
```

结果：

- 自动识别本地文件
- 自动转写
- 自动生成摘要、要点和章节

### 3. 文件夹批量处理

输入：

```text
把这个文件夹里的视频都整理成笔记：F:\素材库\访谈合集
```

结果：

- 自动递归处理文件夹内的媒体文件
- 每个文件单独输出
- 自动生成批处理报告

### 4. 播放列表

输入：

```text
处理这个播放列表并生成摘要：https://www.youtube.com/playlist?list=...
```

结果：

- 每个视频独立输出
- 自动汇总处理结果

## 处理完成后会得到什么

单个条目默认输出到：

```text
outputs/<类型>/<日期>/<名称>/<时间戳>/
```

目录中会有：

- `audio/`
- `transcript/`
- `summary.md`
- `notes.md`
- `chapters.md`
- `knowledge.json`
- `manifest.json`

## 你可以直接这样描述需求

- “帮我把这个视频整理成知识笔记”
- “把这个本地文件转成摘要和章节”
- “把这个文件夹里的视频批量整理成笔记”
- “把这个播放列表全部转写并提炼要点”
