# 中文视频知识抽取器 - 对外目录说明

这是这个 skill 包的公开发布入口顺序说明。

## 一、外部用户优先看什么

1. `README.md`
- 对外首页
- 先看这份，快速知道能做什么、支持什么输入、输出什么结果

2. `examples.md`
- 上手教程
- 直接照着输入就能用
- 适合首次接触这个 skill 的人

## 二、技能触发与能力说明

3. `SKILL.md`
- 这是给 Codex 识别和触发 skill 用的核心说明
- 只保留能力边界、处理原则和核心流程
- 不适合写太多长篇说明

4. `agents/openai.yaml`
- 技能列表里的展示信息
- 决定用户在界面里看到的名称和一句话简介

## 三、执行与实现入口

5. `scripts/process_media.py`
- 统一处理入口
- 负责识别本地文件、文件夹、URL 和批量列表
- 负责下载、转写、长内容分块总结、LLM 后处理和导出

6. `scripts/process_media.bat`
- Windows 下的便捷启动器
- 方便直接运行脚本

## 四、输出格式模板

7. `output_templates/summary.md`
- 摘要页模板

8. `output_templates/notes.md`
- 笔记页模板

9. `output_templates/chapters.md`
- 章节页模板

## 五、发布检查结论

- 入口顺序已经明确
- 中文展示名已经统一
- 示例和 README 已对齐为公开发布风格
- 模板已统一为中文文案
- 本地文件、文件夹、批处理和长视频分块总结都已支持

## 六、建议的对外阅读顺序

1. 先看 `README.md`
2. 再看 `examples.md`
3. 需要给 Codex 直接用时看 `SKILL.md`
4. 需要理解运行逻辑时看 `scripts/process_media.py`
